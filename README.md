# 📌 Overview

CricTech is a modern cricket analytics application designed for cricket enthusiasts, analysts, and developers who want to explore IPL data interactively.

The platform provides:
- 📊 Advanced player statistics
- 🏏 Team-based analytics
- 📅 Season filtering
- 📈 Interactive charts & graphs
- 🔍 Dynamic player search
- 🎨 Modern responsive UI

This project combines **Full Stack Engineering**, **Data Visualization**, and **Sports Analytics** into one scalable platform.

# 🚀 Getting Started

# 1️⃣ Clone the Repository

```bash
git clone https://github.com/hritikr13/crictech.git
cd crictech
```

---

# 2️⃣ Backend Setup (Spring Boot)

## Navigate to Backend

```bash
cd backend
```

## Install Dependencies

```bash
mvn clean install
```

## Run the Backend

```bash
mvn spring-boot:run
```

Backend runs at:

```text
http://localhost:8080
```

---

# 3️⃣ Frontend Setup (Angular)

## Navigate to Frontend

```bash
cd frontend
```

## Install Dependencies

```bash
npm install
```

## Run Angular Application

```bash
ng serve
```

Frontend runs at:

```text
http://localhost:4200
```

# 📡 API Endpoints

## 🎯 Teams API

## 👨‍💻 Players API

## 📅 Filter by Season


